# ACOS v2 Ideas — raw capture

Anything beyond v1 (Phase 18) that isn't a bug or scheduled work. No filtering at capture.
An idea **graduates** to [`ROADMAP.md`](ROADMAP.md) only when there's real demand (a user
asks, or it unblocks a paying use case). Most ideas die here, and that's the point — this is
the parking lot, not a commitment.

## How to file

```
- (area) <idea> — <why it might matter>   [<date>]
```

`area`: same taxonomy as [`../BACKLOG.md`](../BACKLOG.md). Keep it to one line; expand only
on graduation.
