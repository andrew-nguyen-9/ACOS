* Maintainability
* Extensibility
* Traceability

# Mandatory Quality Gates

No module is complete until:

1. Design Complete
2. Implementation Complete
3. Type Checking Passes
4. Security Review Complete
5. Code Review Complete
6. Playwright Tests Added
7. Documentation Updated
8. Commit Generated

---

# Hallucination Prevention Rules

Never invent:

* Metrics
* Dates
* Employers
* Projects
* Certifications

Every generated statement must have evidence.

---

# Definition of Done

The project is complete when:

* Works fully offline
* Uses local models
* Generates resumes
* Generates cover letters
* Generates application answers
* Tracks outcomes
* Learns from outcomes
* Provides ATS analysis
* Provides Career Copilot
* Maintains evidence traceability
* Requires no external AI APIs
