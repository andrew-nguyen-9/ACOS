# AI Career Operating System (ACOS)

## Product Vision

### Mission

Build a local-first AI Career Operating System that maximizes interview rates while maintaining personalization, privacy, traceability, and long-term learning.

The system should continuously improve as more resumes, applications, outcomes, cover letters, projects, GitHub repositories, and question-answer pairs are added.

The application should function independently of Claude or external AI APIs after development is complete.

---

# Guiding Principle

Claude should think like a Principal Software Engineer building a long-lived production product, not a prototype.

Prioritize:

1. Maintainability
2. Extensibility
3. Traceability
4. Testability
5. Reliability
6. Performance

over implementation speed.

---

# Success Metrics

Primary Ranking:

1. Interview Rate
2. Personalization Quality
3. Privacy
4. Generation Speed

The platform should optimize toward interview generation rather than document generation.

---

# User Profile Model

Primary Career Identity:

Senior Analytics & Technology Consultant

Core Experience Areas:

* Litigation Consulting
* Data Analytics
* Financial Investigations
* Product Development
* AI-Assisted Development
* Automation Engineering
* Regulatory Compliance
* Digital Privacy
* Technical Leadership
* Stakeholder Management

---

# Core Philosophy

The system should not be built around resumes.

The system should be built around a Career Knowledge Graph.

Resumes, cover letters, ATS analysis, application answers, interview coaching, and career guidance should all be outputs generated from a shared knowledge system.

---

---

# Learning Philosophy

The platform should improve after every 5 applications.

Update:

* Embeddings
* Rankings
* ATS recommendations
* Experience rankings
* Project rankings

without requiring model retraining.

---

# Hallucination Prevention

Never invent:

* Employers
* Certifications
* Metrics
* Dates
* Projects

All generated content must trace back to evidence.

---

# Evidence Confidence System

Every generated statement receives:

* Verified
* Strong Inference
* Weak Inference

Only verified information may be treated as fact.

---

# Long-Term Vision

The final product should function as:

* Resume Builder
* Cover Letter Builder
* ATS Optimizer
* Career CRM
* Interview Coach
* Personal Career Copilot
* Knowledge Management Platform

while remaining fully local-first and privacy-focused.
